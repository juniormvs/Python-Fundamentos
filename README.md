Este repositório contém exercícios e demonstrações práticas da sintaxe fundamental da linguagem Python. O objetivo é solidificar o domínio sobre as estruturas de dados e o controle de fluxo, elementos cruciais para a manipulação e pré-processamento de dados em projetos de Inteligência Artificial.

## Tópicos Abordados

* `estrutura_de_dados.py`: Demonstração de Listas, Dicionários e Tuplas.
* `controle_de_fluxo.py`: Exemplos de comandos `if/else`, `for` e `while`.
* `funcoes_e_modulos.py`: Implementação de funções básicas e uso de módulos padrões.
